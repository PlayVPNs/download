**PlayVPN** — это кроссплатформенное решение для обеспечения анонимности и высокоскоростного доступа к заблокированным ресурсам. Наша сеть включает более 500 серверов в 60 странах мира.

## 🚀 Основные возможности
* **AES-256 Encryption:** Шифрование военного уровня.
* **No Logs Policy:** Мы не храним историю ваших подключений.
* **Auto-Connect:** Автоматический выбор самого быстрого сервера.
* **Kill Switch:** Защита данных при обрыве соединения.

## 📥 Установка (Windows)

Для начала работы необходимо скачать и запустить установщик.

1. Перейдите в папку `playvpn/`.
2. Запустите файл **`PlayVPN.exe`**.
3. Следуйте инструкциям мастера установки.

> [!IMPORTANT]
> Для активации Beta-доступа в процессе установки вам потребуется ключ.
> Ключ находится в файле: `playvpn/README.md`
> **КЛЮЧ:** `key_p14yvpn`
